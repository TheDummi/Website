DummiBot

About

DummiBot is an easy to set up fun bot. It also includes Moderation, emotion and action commands.
What makes DummiBot special? DummiBot is your standard all in one bot. However, it does have random
reactions, this is a command that not a lot of bots have, Dummi has it! Ofcourse we have way more
than that.

Setup/Important info before you invite DummiBot.

To setup DummiBot use `~setup` This will give you a setup menu. Just follow the steps as it says
the embed.Try to keep the `embed links` permission on at all times else the bot won't work!
This also comes with the basic permissins for bots in order to work.

In case you want to know the prefix of the current server than run `@DummiBot#2349`. By default this is `~`.

For any more questions or bug reports join our support server `https://discord.gg/AJxtuxg`.
