About DummiBot

DummiBot is a Fun and Easy to use Discord bot. Its main
purpose is to give you and your server members a fun and
easy time.

Why DummiBot? What does it have what others don't?

DummiBot Doesn't have too many special commands that
other bots don't have.
However it does have random reactions. Random reactions
allows the bot to add a random reaction to your messages
once in a while.

In order to turn this on run ~setup

Invite DummiBot

You can invite DummiBot with a lot of different permissions
enabled/disabled. To keep the site clean and easy to use we include 2 ways which
guarantee you to have all commands work properly.
The first way is with administrator permission, this basically
means that the bot has full access to all permissions Discord offers.
This also guarantees you to have the bot work no matter what! Any
update/new feature will guarantee work from the moment it gets added.

However, if you are the kind of person who doesn't like to have
people/bots to have the administrator permission. You can always
invite DummiBot to have the needed permissions in order for it
to work. Some features will look different as supposed to how
they're supposed to look.

  • If you want to invite DummiBot with the administrator permission,
    click this.
  • If you don't want DummiBot To have the administrator permission,
    click this.
  • If you want more options on what invite permissions you want,
    than join a server with DummiBot in it or our support server
